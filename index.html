Spa RFM analysis

This analysis focuses on utilizing RFM (Recency, Frequency, Monetary) analysis to delve into client behavior and create effective strategies for re-engaging clients in a spa environment. By analyzing factors such as how recently a client visited, how frequently they visit, and how much they spend, this approach aims to gain insights into client preferences and patterns. The analysis consists of two files, a Jupyter notebook, and a .py file that contains all the functions used for data cleaning and data visualization.
